Урок 8. Экспорт данных из Excel в формат XML
 

3. Метод преобразования

 

Перед экспортом данных требуется определить требования к формату XML-документа – узнать имена тегов, их структуру, наличие атрибутов. После этого на лист книги Excel можно добавить несколько строк, в которых будут помещены имена основных элементов, которые находятся в табличной части документа. Также потребуется провести преобразование некоторых символов, которые не могут использоваться в информационном содержании XML-документа (например & “ ‘ < >) и будут заменены сущностями (&amp; &quot; &apos; &lt; &gt;) При конкатенации ячеек потребуется использовать следующие функции Excel:

•  ЕПУСТО – для проверки «пустоты» ячейки, требуется для определения элемента «родительского» уровня

•  ЕСЛИ – для формирования условия с использованием функции ЕПУСТО для определения родительского или дочернего элемента.

•  ПОДСТАВИТЬ – для замены текста в ячейках – требуется для замены некоторых символов на сущности.

Перед работой с Excel-книгой потребуется установить формат представления адресов ячеек в виде «A1». Для этого потребуется в меню Сервис выбрать пункт Параметры и на вкладке Общие убрать отметку с флажка Cтиль ссылок R1C1.

При проведении конкатенации ячеек можно использовать абсолютную адресацию (символ $) для ссылки на ячейки с именами элементов, символов для замены и сущностей.

Во всех остальных случаях можно использовать относительную адресацию – без символа $ При использовании такой адресации упрощается процедура распространения формул по нужным строкам.

До проведения конкатенации потребуется выделить блок ячеек, в которых содержится нужная информация, удалить из этого блока пустые строки и (возможно) те товары, которые не требуется отображать в прайс-листе. Для этого можно использовать фильтрацию (Автофильтр).

Возможна ситуация, при которой потребуется получить уникальный номер для каждого товара. Для этого можно использовать функцию Excel СТРОКА().

В Практической работе приведен пример формирования XML-документа на основе прайс-листа в формате Excel, сформированного программой 1С «Торговля и склад».
