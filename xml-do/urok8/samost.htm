Урок 8. Экспорт данных из Excel в формат XML
 

Самостоятельная работа

 

Задание. На основе книги Excel, сохраненной в файле Магазин.xls, требуется создать документ XML со структурой, показанной на рисунке:

 

Пояснения к структуре документа:

  • Значением атрибута date должна быть текущая дата в формате дд.мм.гггг
  • Значением атрибута npp должно быть целое число, уникальное для каждого товара.
  • Значением атрибута price должна быть отпускная цена товара с точностью «до копеек».

XML-документ должен быть сохранен в файле с именем price_firma.xml, который Вы должны прислать преподавателю на проверку.
