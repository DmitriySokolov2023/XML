Самостоятельная работа

 

Задание 1. Создать XML-документ с указанной структурой при помощи программы XMLPad c проверкой структуры.

 

Информационное содержание элементов показано в таблице

Элемент

Содержание

NAIM1

Сахар

NAIM2

Макароны

VID

Продукты

 

Имя файла для сохранения документа – 1.xml

Задание требуется выполнить в программе XMLPad и в связке текстовый редактор Notepad - браузер Internet Explorer. Кодировка документа должна обеспечивать показ русских букв.

 

Задание 2. Исправить ошибки в документе следующего содержания:

</TOVАR>

</vid>

Продукты

<vid>

<0nazv>

Вода минеральная

<price>

200

</price>

</0NAZV>

</TOVAR>

Подсказка: При анализе файла обратите внимение на правильность написания тегов и наличие русских букв в именах тегов. Пролог документа добавить самостоятельно. Кодировка должна обеспечивать отображение русских букв.

Исправленный документ требуется сохранить в файле с именем 2.xml.

 

Задание 3. Контрольные вопросы:

•  Может ли конечный тег начинаться с точки?

•  Может ли начальный тег начинаться с цифры?

•  Сколько корневых элементов должно быть в документе XML ?

•  Какие кодировки обеспечивают отображение русских букв?

•  Где указывается кодировка документа и версия языка XML ?

•  Можно ли для создания XML документа использовать редактор Microsoft Word?

•  Чем отличается начальный тег элемента от конечного?

•  Какая программа используется для просмотра XML документа без возможности изменения?

•  В каком формате сохраняется XML документ?

Ответы на вопросы оформите в виде файла в формате Word и сохраните под именем otvet.doc.

Файлы 1.xml, 2.xml и otvet.doc пришлите преподавателю на проверку.
