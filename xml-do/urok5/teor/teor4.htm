4. Задание типов элементов в схеме XML

 

Одним из важнейших достоинств схем XML является задание типа информационного содержимого элемента. При задании типа содержимого можно жестко задать тип данных, которые передаются при помощи этого элемента. Типы задаются в атрибуте type. Возможные типы элементов перечислены в таблице:

Тип элемента

Пояснение

Пример содержимого

String

Строковый

“название”

Decimal

число с дробной частью

10.21

Integer

Целочисленное значение

100

nonNegativeInteger

Целое неотрицательное значение

0;1234

Date

Дата в формате гггг-мм-дд

2005-01-31

Time

Время в формате чч:мм: cc

21:59:59
