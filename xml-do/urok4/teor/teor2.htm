2. Задание пространства имен

 

Для обеспечения уникальности требуется задать специальный параметр с уникальным значением. Пространство имен задается парой значений – префиксом (на примере выделен подчеркиванием) и URL -адресом (в формате http ://www.). URL -адрес должен быть уникальным, обычно указывается адрес сайта компании, этот адрес необязательно должен быть существующим – можно задать вымышленный, но он должен обеспечивать уникальность – на случай будущего объединения данного документа с другим документом. Префикс задается во всех элементах, которые принадлежит этому пространству имен. Пространство имен задается в начальном теге элемента с использованием служебного слова xmlns Пример задания пространства имен в корневом элементе документа:

<?xml version="1.0" encoding="windows-1251" ?>

<TVR:TOVARS

xmlns:TVR=”http://www.firma-tovar.ru”>

>  

…

</TVR:TOVARS>

В этом документе задано пространство имен с URL-адресом: http://www.firma-tovar.ru, а префикс будет задан с именем TVR Требуется обратить внимание на то, что этот префикс, с двоеточием предшествует всем именам элементов данного пространства имен, как в начальном, так и в конечном тегах. Дополнительно можно задать несколько пространств имен, которые будут использованы в элементах этого документа.

<?xml version="1.0" encoding="windows-1251" ?>

<TVR:TOVARS

xmlns:TVR=”http://www.firma-tovar.ru” 

xmlns:PRZ=”http://www.catalog-firm.ru”

>  

…

</TVR:TOVARS>

Первое пространство имен (http://www.firma-tovar.ru) с префиксом TVR будет использоваться для описания товаров, а второе (http://www.catalog-firm.ru) с префиксом PRZ для описания производителей товаров.

После преобразования документ будет выглядеть следующим образом:

<?xml version="1.0" encoding="windows-1251" ?>

<TVR:TOVARS

xmlns:TVR="http://www.firma-tovar.ru"

xmlns:PRZ="http://www.catalog-firm.ru">

<TVR:TOVAR>

<TVR:NAIM>C ыр </TVR:NAIM>

<TVR:VID> Продукты </TVR:VID>

<TVR:PRICE>420</TVR:PRICE>

<PRZ:PROIZV>

<PRZ:NAIM>Cырзавод №124</PRZ:NAIM><PRZ:CITY> С - Петербург </PRZ:CITY>

</PRZ:PROIZV>

</TVR:TOVAR>

<TVR:TOVAR>

<TVR:NAIM> Бензин </TVR:NAIM>

<TVR:VID> Горючее </TVR:VID>

<TVR:PRICE>20</TVR:PRICE>

<PRZ:PROIZV>

<PRZ:NAIM> НПЗ №222</PRZ:NAIM><PRZ:CITY> С - Петербург </PRZ:CITY>

</PRZ:PROIZV>

</TVR:TOVAR>

</TVR:TOVARS>

При использовании пространств имен в этом документе выполнено разделение пространств имен элементов: элементы, относящиеся к товарам отделены от элементов, относящихся к производителям. Помимо этого, документ стал более читабельным – можно с первого взгляда определить вид информации, которая содержится в каждом элементе.

В документе можно задать и пространство имен по умолчанию. Обычно это требуется в том случае, когда многие элементы документа описывают объекты одного вида и всего лишь несколько элементов описывают объекты другого вида. В заданном документе пространством по умолчанию может служить пространство имен товаров. Для задания пространства имен по умолчанию, применяется другой синтаксис, чем для именованного пространства имен – префикс отсутствует. Соответственно и при определении пространства имен не ставится префикс с двоеточием. Применяется следующая запись:

<TVR:TOVARS xmlns=”http://www.firma-tovar.ru”>

При использовании пространства имен по умолчанию, вид документа будет следующим:

<?xml version = "1.0" encoding = "WINDOWS-1251"?>

<TOVARS

xmlns="http://www.firma-tovar.ru"

xmlns:PRZ="http://www.catalog-firm.ru">

<TOVAR>

<NAIM>Cыр</NAIM>

<VID>Продукты</VID>

<PRICE>420</PRICE>

<PRZ:PROIZV>

<PRZ:NAIM>Cырзавод №124</PRZ:NAIM><PRZ:CITY>С-Петербург</PRZ:CITY>

</PRZ:PROIZV>

</TOVAR>

<TOVAR>

<NAIM>Бензин</NAIM>

<VID>Горючее</VID>

<PRICE>20</PRICE>

<PRZ:PROIZV>

<PRZ:NAIM>НПЗ 222</PRZ:NAIM><PRZ:CITY>С-Петербург</PRZ:CITY>

</ PRZ : PROIZV >

</ TOVAR >

</ TOVARS >

Видно, что в этом случае упрощается структура документа.

Для обработки XML -документа часто используются зарезервированные пространства имен, например –

xsd =" http :// www . w 3. org /2001/ XMLSchema ">

Это пространство имен задается в XML-схемах, о которых будет рассказано на следующем уроке.
